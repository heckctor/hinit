## Empezando

```bash
git clone https://github.com/heckctor/hinit.git
```

### Prerequisitos

Tener instalado:

* node
* npm ó yarn
* gulp de manera global 

```bash
npm install gulp-cli -g
```


### Instalación de dependencias

```bash
npm install
```

ó

```bash
yarn
```

### Iniciar Servidor de prueba
escribir en la terminal el comando

```bash
gulp
```
